ТРЕБОВАНИЯ



  1. Как только страница Редактора открыта, все поля, включая поле изображения, должны быть незаполненными;
  2. Как только страница Редактора открыта, кнопка Готово должна быть задисейбленной;
  3. Сразу после клика по любому месту страницы, например если любое из полей получает фокус, страница переходит в состояние, отображенное на е02.html, то есть все поля заполнены;
  4. Как только страница Редактора переходит в состояние е02.html, кнопка Готово должна заинейблиться;
  5. Клик на активную кнопку Готово закрывает страницу Редактора с переходом на предыдующую страницу.
