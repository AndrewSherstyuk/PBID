ХЕДЕР
  1. Лого PencilBOOM должно вращаться либо не вращаться в зависимости от того, в каком состоянии находится кнопка главного меню Остановить/Запустить Лого;
  2. При скролле страницы вниз, лого PencilBOOM и кнопка главного меню должны быть зафиксированы. Пример здесь pencilboom.ho.ua/newversion/index.html;
  3. Клик по лого или названию PencilBOOM должен редиректить на Конкурсариум;
  4. Клик по пункту Конкурсариум должен редиректить на Конкурсариум;
  5. Клик по пункту Гениариум должен редиректить на Гениариум;
  6. Клик по пункту О Нас должен редиректить на страницу О Нас;
  7. Поле поиска: значок линзы должен быть кликабельным и очищать поле поиска;
  8. Поле поиска должно поддерживать юзер инпут без валидации. Очищаться должно по клику на линзу;
  9. Дроп даун выбора языка должен экспандиться по клику;
  10. Дроп даун выбора языка: выбранное значение должно применяться только к дроп дауну. По факту язык демо версии только русский;
  11. Дроп даун выбора языка: при экспанде стелка направлена вверх, при коллапсе - вниз;
